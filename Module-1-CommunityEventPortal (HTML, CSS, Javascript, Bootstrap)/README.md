# Module 1 - Community Event Portal

## Overview

This project was developed as part of the Cognizant Digital Nurture (DN 5.0) Java FSE Upskilling Program.

The Community Event Portal is a browser-based web application that combines concepts from HTML5, CSS3, JavaScript, and Bootstrap 5. The project demonstrates web page structuring, navigation, forms, event handling, styling, responsive design, DOM manipulation, asynchronous operations, and Bootstrap-based user interface development.

The application allows users to browse community events, view event information, interact with event-related content, and submit registrations through an interactive web interface.
