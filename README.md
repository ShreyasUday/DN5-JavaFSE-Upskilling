# Cognizant (DN5 Java FSE Upskilling)

## Overview

This repository contains solutions and project work completed as part of the Cognizant Digital Nurture (DN 5.0) Java Full Stack Engineer Upskilling Program.

The repository currently includes:

* Module 1 - Community Event Portal (HTML5, CSS3, JavaScript, Bootstrap 5)
* Module 2 - ANSI SQL Using MySQL
* Module 3 - Core Java

Each module contains its own source code, resources, and README documentation.
